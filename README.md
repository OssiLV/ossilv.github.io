# 🌐 Personal Portfolio | Blog | Courses

## 🇻🇳 Giới thiệu

Đây là repository của trang **portfolio cá nhân**, được xây dựng với mục tiêu chia sẻ kiến thức và giới thiệu bản thân. Website bao gồm:

- 🎯 **Trang giới thiệu cá nhân**: Thông tin, kỹ năng, dự án cá nhân.
- ✍️ **Blog**: Viết và chia sẻ kiến thức về công nghệ, kinh nghiệm làm việc.
- 📚 **Courses**: Tổng hợp và chia sẻ các khóa học tự xây dựng hoặc đã học.

Trang web được thiết kế với phong cách **tối giản, hiện đại**, tối ưu cho cả desktop và mobile.

## 🇺🇸 Overview

This repository contains a **personal portfolio website** built to showcase personal information, share knowledge, and host learning content. It includes:

- 🎯 **Portfolio**: About, skills, and personal projects.
- ✍️ **Blog**: Writing about tech, experience, and tutorials.
- 📚 **Courses**: Learning materials and self-created content.

The site features a **minimalist, modern design**, responsive across devices.

## 🚀 Công nghệ | Technologies

- [Astro](https://astro.build/) – Static site generator
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework
- [Markdown](https://www.markdownguide.org/) – Content writing (blog, course)
- [TypeScript](https://www.typescriptlang.org/) – Optional typing (nếu dùng)
